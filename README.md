# Rulebook

This is a standalone branch used to hold reference material for the **mech_foundry**
Foundry VTT system — specifically the *A Time of War* core rulebook used as the
authoritative source for game mechanics.

This branch is intentionally kept separate from the code branches (`main` and the
`claude/*` feature branches). It shares **no git history** with them and contains
**no system code** — only the rulebook reference file(s).
